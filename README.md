# HackMySber - step-by-step guide to unlocking your android based SberDevice.

# REQ.

1. a SberDevice or a super-restricted android based device
2. know what "rooting" "adb" "shell" is
3. Reading.
4. and a vpn if youre in russia.

# DEPENDENCIES

ADB TV (for tv boxes) or ADB APP CONTROL (phone/tablet)
Shizuku
Key Mapper
Your favourite way to send files to the SberDevice
A Launcher (like projectivy)

ALL THE DEPENDENCIES ARE IN THE "dependencies" RELEASE!

# Guide for SberBox (2) or any other device

STEP 1

firstly, enable developer options and usb debugging and wireless debugging

STEP 2

setup ADB TV/APP CONTROL, when you get into the adb frontend, go to shell and run this command to set the custom launcher:

pm set-home-activity *your launchers package name*/.MainActivity

STEP 3

go to apps, then disable the stock launcher.

STEP 4

setup shizuku

STEP 5

get keymapper and map the bloatware buttons to custom stuff

STEP 6 

youre done)
